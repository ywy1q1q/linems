# LINEBOT youbike :bike:
### 查詢附近站點、數量、可停數量
身為一個皮克敏玩家，騎腳踏車是必備重要工具，只須站在原地就可以顯示附近站點，\
以及附近站點可租借的腳踏車數量，可停空位。\
而公共自行車有分兩種車輛，有2.0一般車及2.0E電輔車，\
有些並不會將數字分類為兩種，是將數字家在一起，\
偏好有固定騎的車種可以看此數量，來知道怎麼選擇站點。

## 使用方法

可文字輸入 **'找車'**，即可顯示要求傳送目前位置資訊，\
機器人會為您搜尋附近3站。


也可直接按下方圖文選單來去做傳送 :round_pushpin: 目前位置資訊，\
下方圖文選單也可按使用說明，來引導網址至youbike官網，\
可以在裡面觀看使用方法，及註冊、保險。
